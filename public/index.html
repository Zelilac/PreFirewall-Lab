üöÄ Quick Start

Ready to test? Visit the Interactive Attacks Page to try vulnerabilities in your browser!

Using curl (Command Line)

# SQL Injection
curl "http://localhost:3000/api/sql/users?username=admin' OR '1'='1"

# XSS
curl "http://localhost:3000/api/xss/search?q=<script>alert('XSS')</script>"

# Command Injection
curl -X POST http://localhost:3000/api/command/ping \
  -H "Content-Type: application/json" \
  -d '{"host": "127.0.0.1; whoami"}'

# Path Traversal
curl "http://localhost:3000/api/traversal/download?file=../../../../etc/passwd"

Using Postman

  1. Import postman/PreFirewallLab.postman_collection.json
  2. Set variable baseUrl to http://localhost:3000
  3. Run any request to test vulnerabilities
